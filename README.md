# 🍛 Tasty Plate - The Legend of Old Dhaka

**Tasty Plate** is a modern restaurant application dedicated to bringing the authentic heritage flavors of Old Dhaka (Kacchi, Tehari, Borhani) to the digital world. This project features real-time database integration, secure user authentication, and a unique **AI Chef Chatbot** that provides personalized food recommendations.

## ✨ Features

- **🔥 Dynamic Menu:** Fetches menu items in real-time from Firebase Firestore.
- **🛒 Smart Cart System:** Seamless functionality to add items to the cart and place orders.
- **🤖 AI Chef Chatbot:** Powered by Google Gemini AI, this chatbot acts as a witty Old Dhaka expert to recommend dishes based on user mood.
- **🔐 User Authentication:** Secure Login and Signup system using Firebase Authentication (Email/Password).
- **📱 Responsive Design:** Fully optimized for Mobile, Tablet, and Desktop devices.
- **🏷️ Category Filters:** Easy filtering for Breakfast, Lunch/Dinner, and Drinks.

## 🛠️ Tech Stack

This project is built using:

- **Frontend:** React.js, TypeScript, Vite
- **Styling:** Tailwind CSS
- **Backend (BaaS):** Firebase (Firestore Database, Authentication)
- **AI Integration:** Google Gemini API (`@google/generative-ai`)
- **Icons:** Lucide React
